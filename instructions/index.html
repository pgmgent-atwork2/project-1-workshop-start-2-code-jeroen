Workshop - Start To Code

We gaan een spelletje Tic Tac Toe maken!

Daarvoor hebben we drie codetalen nodig

  • HTML
  • CSS
  • JavaScript

Geen paniek, we houden de code eenvoudig, maar maken wel iets interactief!

STAP 1 - MAPPEN EN BESTANDEN

We hebben een code-editor nodig. In de workshop gebruiken we Visual Studio Code.

  • 1.1 Maak een nieuwe map aan (vb op je bureaublad) en geef als naam TicTacToe
  • 1.2 Open het programma Visual Studio Code. en open de folder die je zopas aangemaakt hebt.
  • 1.3 Bouw onderstaande mappen- en bestandenstructuur aan
  • 1.4 Installeer de live server extensie in Visual Studio Code
  • 1.5 Start de live server door rechtsonder de blauwe balk te klikken op Go Live
    Jouw standaard browser zal nu gestart worden en je zal een blanco webpagina zien!
  • Goed gedaan!

STAP 2 - HTML & CSS

Tijd om te coderen, zodat we meer te zien krijgen dan een witte webpagina!

  • 2.1 Kopieer onderstaande code in jouw index.html-bestand. Bewaar vervolgens dat bestand.
    Dit is een basis-structuur waarin we verder werken.
  • 2.2 Kopieer onderstaande code in jouw main.css-bestand. Bewaar vervolgens dat bestand.
    Dit bevat alvast een aantal basisstijlen die we later zullen nodig hebben voor de look & feel van onze app.
  • 2.3 Bekijk het resultaat in de browser (de url is hoogstwaarschijnlijk: http://127.0.0.1:5500/).
    Als alles goed is, ziet het er nu zo uit. Een volledig gekleurd vlak.
  • 2.4 Laten we het eens van kleur veranderen. Hiervoor moeten we de kleurwaarde veranderen. Zoek naar background-color: orange; en verander orange; naar bijvoorbeeld darkslateblue; De achtergrond is nu blauw-paarsachtig.
  • 2.5 We hebben nu wat inhoud nodig voor onze webpagina.
    Voeg onderstaande code toe, en plaats ze tussen de openings <body>-tag en sluitings </body>-tag Je zal nu een Tic Tac Toe raster zien verschijnen.

STAP 3 - INTERACTIVITEIT MET JAVASCRIPT

Nu moeten we nog zorgen dat we het spelletje ook kunnen spelen.

  • 3.1 Eerst en vooral moeten we de link leggen naar het JavaScript-bestand.
    Voeg deze regel code toe, onderaan net voor de sluitings-body-tag.
    <script src="./scripts/app.js"></script>
  • 3.2 Ga naar het bestand app.js en zet er het volgende in:
    alert('Hello world')
    Bewaar het bestand en kijk wat er in de browser gebeurt (als je deze herlaadt)
  • 3.3 Als het werkt, mag je de regel terug wissen, we weten nu dat ons JavaScript-bestand uitgevoerd wordt :)
  • 3.4 Kopieer onderstaande code naar het app.js bestand. In deze code maken we een array die de ingevulde waarden van het speelveld bijhoudt.
    Alle waarden starten op null, dit betekent dat er geen waarde is toegekend. We leggen ook vast dat de actieve speler Speler 1 is.
  • 3.5 Plak nu het volgende stukje code onder de al aanwezige code in het app.js bestand.
    De eerste functie zorgt ervoor dat het spel zal registreren als je in 1 van de vakjes klikt. De 2de functie zal dan een waarde "X" of "O" toekenen aan de array die we daarnet gemaakt hebben.
    Je kan nakijken of dit werkt door in je browser rechts te klikken en inspect te kiezen. Er zal een schermpje openen aan de rechterkant van je browser. Als je daar op Console klikt, dan zal je de array zien met de toegekende waarde.
  • 3.6 We willen uiteraard de X'en en O's in ons raster zien verschijnen. De volgende code zal hier voor zorgen.
    Plak deze code onder de setGridValue functie. Vervang in de setGridValue functie de console.log(gridValues); lijn door fillGridHTML(gridValues);
  • 3.7 Het spel werkt nu. Maar het is niet echt eenvoudig om te zien wie er gewonnen heeft. Door volgende code toe te voegen kunnen we dit oplossen.
    Deze code zal steeds controlleren of er iemand gewonnen heeft en een melding laten verschijnen wanneer dit het geval is.
